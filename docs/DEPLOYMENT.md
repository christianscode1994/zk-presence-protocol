Vercel deployment steps (concise)

1. Push this repo to GitHub.
2. Create a new Project in Vercel, connect the GitHub repo.
3. Add Environment Variables in Vercel:
   - RELAYER_API_KEY = (secure random value)
   - NEXT_PUBLIC_APP_NAME = "ZK Presence Prototype"
   - NEXT_PUBLIC_SUPPORT_EMAIL = "security@example.com"
4. Deploy. The site will appear at your Vercel project URL.
